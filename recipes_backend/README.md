# postgresql db - express api for recipes microservice

## RECIPE DB

- postgresql in docker container
- knex.js
- recipes schema:
  - id
  - ingredients []
  - name
  - decriptions
  - instructions
  - categories []
  - prep_time
  - cook_time
  - wait_time
  - servings
  - calories
  - fat
  - carbs
  - fiber
  - sugar
  - protein
- seed db with recipes daily
- scrape web for recipes
